# 🏛️ ElectionAI · 선거 후보자 비교 분석 플랫폼

**AI 기반 선거 후보자 종합 분석 시스템**

## 🎯 주요 기능

### 1️⃣ **3단계 순차 분석 파이프라인**
```
01. 통합분석리포트
   ↓ (강점/약점 + 경쟁구도)
02. 선거 판세 인텔리전스
   ↓ (판세 분석 + 승패변수)
03. 당선 전략 설계
   → (전략 설계 + 90일 로드맵)
```

### 2️⃣ **후보별 독립 분석**
- 각 보고서 후보별 독립된 탭 구조
- 후보별 개별 PDF 다운로드 기능
- 파일명에 선거구 자동 표기

### 3️⃣ **AI 기반 분석**
- Claude, ChatGPT, Gemini, Grok 4개 모델 점수 비교
- 모델별 강점 분석 및 결과 종합

### 4️⃣ **실행 중심 보고서**
- 분석만 제공하지 않고 실행 전략까지 제시
- 90일 액션 로드맵 및 캠프 조직별 분배표
- 리스크 대응 매뉴얼

## 🚀 사용법

### 1. 홈페이지 접속
[홈으로 이동 →](index.html)

### 2. 초기 설정
1. **Google Gemini API 키 설정** - [Google AI Studio](https://aistudio.google.com/app/apikey)에서 발급
2. **선거구 & 선거유형 입력** - 분석 대상 지역 정보 입력
3. **PDF 파일 업로드** - PoliticianFinder.com 평가 리포트 (최대 5개)

### 3. 분석 시작
- "분석 시작" 버튼 클릭
- 후보자 점수 및 평가 자동 추출

### 4. 보고서 생성
- **01단계** · 통합분석리포트로 종합 평가 확인
- **02단계** · 선거 판세 인텔리전스로 경쟁 구도 분석
- **03단계** · 당선 전략으로 맞춤형 전략 수립

### 5. PDF 다운로드
- 각 보고서 우측 상단 "PDF 저장" 버튼
- 후보별/단계별 개별 다운로드 가능
- 파일명에 선거구 자동 표기

## 📊 분석 지표 (10개)

1. **공익성** - 공공 이익 추구 정도
2. **정책실행력** - 공약 실행 능력
3. **전문성** - 해당 분야 전문 지식
4. **대응성** - 민원/이슈 대응 능력
5. **책임성** - 책임감 및 투명성
6. **소통능력** - 국민과의 소통 능력
7. **리더십** - 리더십 및 추진력
8. **비전명확성** - 미래 비전 제시 정도
9. **청렴성** - 청렴도 및 도덕성
10. **윤리성** - 윤리적 기준 준수
